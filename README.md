# OlympusGCP-Firebase

Unified application backend cluster for the Olympus fleet, hosting the Firebase suite.

## Purpose
Supports the **OlympusVision** (Flutter) front-end with:
1.  **Authentication**: Identity management and custom claims.
2.  **Firestore**: Application-state database.
3.  **Functions**: Ephemeral serverless logic.
4.  **Storage**: Object storage for UI assets and multi-modal data.

## Tooling
This workspace is optimized for the **Firebase CLI**. Run all firebase commands from the root of this workspace.
